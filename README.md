# to-do

### Desafio:

Desenvolver uma aplicação de Lista de Tarefas onde é possivel criar, listar, editar e deletar as tarefas individualmentes.

A aplicação deve consumir uma API que armazenas as listas de tarefas.

#### Telas:

- Página inicial: uma tabela com as tarefas, com as colunas de Titulo, Descrição e Ações (Editar e Deletar), além disso deve haver um botão para criar novas tarefas que abre uma nova página de criação de tarefas. Por sua vez o botão de Editar deve abrir uma nova página de edição de tarefas e o de Deletar deve exibir uma mensagem de confirmação e caso confirmado a tabela deve ser atualizada e deve haver um feedback para o usuário.
- Página de criação de tarefa: é um formulário com os campos de Titulo e Descrição e um botão para concluir. O formulário deve ter validação e feedback para o usuário.
- Página de edição de tarefa: é um formulário com os campos de Título e Descrição já preenchidos e um botão para concluir. O formulário deve ter validação e feedback para o usuário.
